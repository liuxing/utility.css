## Flex

### Horizontal alignment

  • .flex-left Add this class to align flex items to the left
  • .flex-center Add this class to center flex items along the main axis.
  • .flex-right Add this class to align flex items to the right.
  • .flex-between Add this class to distribute items evenly, with equal space between the items along the main axis.
  • .flex-around Add this class to distribute items evenly with equal space on both sides of each item.
